WSVideoPlayer
=============

A convenient video player for iOS5.0+.   
WSVideoPlayer is mainly developed with two ways, they are MPMoviePlayerViewController and AVFoundation.   
At the meantime, there is a HTML5 <video> demo in this project.   
PLS ENJOY IT!
